# Dlubal Container Test

## Prerequisites

- Install [Docker Deskop for Windows](https://hub.docker.com/editions/community/docker-ce-desktop-windows).
- Switch to Windows containers.

## Run Console App

```
dotnet run
```

## Build Docker Image

```
docker build -t dlubal-test .
```

## Run Docker Image

```
docker run dlubal-test
```
